# Welcome to Markdown Vault 👋

This note lives in a **real folder** on your device. Open the same folder in **Obsidian** and you'll see identical files — Markdown Vault never locks your notes away.

![[Attachments/banner.png]]

## ✍️ Write with live preview

Switch between **Edit**, **Split**, and **Preview** from the toolbar. Formatting works the way you expect:

- **Bold** and *italic*
- `inline code`
- [Links](https://www.tertiaryinfotech.com)
- > Block quotes for the important bits

## 🖼 Images, Bear-style

Use Obsidian embeds `![[image.png]]` or standard Markdown `![alt](path.png)` — both render inline:
