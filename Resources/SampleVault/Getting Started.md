# Getting Started

1. Tap **Open Folder…** and choose any folder of Markdown files.
2. Or keep using this **Sample Vault** to explore the features.
3. Create notes and folders with the **+** button.
4. Everything is saved as plain `.md` files — back them up however you like.

## Keyboard (Mac)

- `⌘N` — new Markdown file
- `⌘O` — open a vault folder

Enjoy writing! ✨
